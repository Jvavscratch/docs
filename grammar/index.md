## 变量和赋值

所有变量默认是全局的。如果变量和列表同名，sb3 将无法正确构建。

```js
let x = 3; // 有效
let y = x; // 有效

y = 2 * 3; // 有效

// 引用名为 'example' 的列表（_list_ 将被省略）
let z = _list_example

// hello_world.js

let myStr = "Hello, World!";
looks.say(myStr);
```

您可以通过在名称前加上 `_l_` 来强制变量为私有：
```js
let _l_test = 5; // 局部变量 "test"
```

如有需要，可以通过在名称前加上 `_g_` 来创建全局变量：
```js
let _g_test = 5; // 全局变量 "test"
let _g__l_test = 5; // 全局变量 "_l_test"
```

虽然当前已禁用，但可以通过在名称前加上 `_c_` 来创建云变量：
```js
let _c_test = 5; // 云变量 "test"
```

> **警告**：云变量当前已禁用。它们将被转换为 `global` 变量，并且 `_c_` 前缀将被移除。

> **注意**：命名云变量、全局变量或局部变量使用相同名称将导致构建失败。

这些变量可以像普通变量一样引用：
```js
let _l_private = "private variable";
private = "test";
```
